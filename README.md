# Python 变量和数据类型

## 简介

展示 Python 的基本数据类型：整数、浮点数、字符串、布尔值、列表、元组、集合、字典。

## 启动

```bash
python variable_types.py
```

## 教程

### 基本类型

- **int**: 整数
- **float**: 浮点数
- **str**: 字符串
- **bool**: 布尔值

### 容器类型

- **list**: 列表 (可变)
- **tuple**: 元组 (不可变)
- **set**: 集合
- **dict**: 字典

### 类型转换

```python
int("42")     # 字符串转整数
float("3.14") # 字符串转浮点数
str(42)       # 整数转字符串
```
